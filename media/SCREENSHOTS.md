## How to Create Screenshots

1. Open VS Code with the Explorer 2 extension loaded
2. Set up Panel I to point to one folder (e.g., project files)
3. Set up Panel II to point to another folder (e.g., tests or docs)
4. Use VS Code's built-in screenshot tool or external tools:
   - macOS: Command+Shift+4 then Space to select window
   - Or use Cmd+Shift+F5 in VS Code activity indicator

## Format Requirements

- Format: **PNG** (lossless quality important)
- Size: 800x600px or similar 4:3 ratio
- No extra UI elements (taskbars, other windows)
- Clear, well-lit content
- Professional appearance
